Programming in Ocaml

Ocaml is a high-level, yet highly efficient programming language which supports both object-oriented and functional programming styles. We refer you to the faq for some background on this language and why we chose it.

Documentation

We recommend having at least basic familiarity with ocaml prior to starting out in nab. There are many great resources (Books and Tutorials) available to learn ocaml; two excellent ones in particular are:
  • The 'Oreilly book' (online in English or in French) is an well-written and well-structured book, with both introductory and advanced material. (a good read through chapters 2, 3, and 4 should be sufficient to get started in nab).
  • A more practical approach is in R. Jones' tutorial, which teaches ocaml by comparing and contrasting it with c and java.

Community

A lot of valuable information can be obtained through the ocaml mailing lists.
Ocaml_beginners is a great place to ask questions when starting out, and the caml-list is also a valuable resource for less trivial problems.
