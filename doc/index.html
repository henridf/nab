NAB (a Network in A Box)

NAB is a network simulator targeted at wirelss sensor and ad hoc networks. It is developed within the MICS project at EPFL.

NAB is written in ocaml, a high-level yet highly efficient programming language which supports both object-oriented and functional programming styles.

NAB places a particular emphasisis on scalability (simulate tens of thousands of nodes on commodity hardware), visualization (a few pics here), and a clean, flexible architecture which was designed from scratch for the purpose of simulating large-scale networks.

Routing protocols currently implemented in NAB are AODV, EASE, and FRESH. Various MAC layers are available including a null MAC and a CSMA MAC. A 802.11b MAC is under development.
